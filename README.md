# Stock_analysis

Stock Market Performance Analysis — 2024
An end-to-end analysis of 5 major stocks (AAPL, GOOGL, AMZN, TSLA, MSFT) over the full year 2024, built across two tools to demonstrate a complete data analyst workflow.

Project Overview
DetailInfoDatasetSimulated daily stock prices — 252 trading daysStocksApple, Google, Amazon, Tesla, MicrosoftPeriodJanuary – December 2024ToolsPython (Pandas, Matplotlib) · Microsoft Excel

Key Questions Answered

Which stock delivered the best total return over the year?
Which stock had the best risk-adjusted return (Sharpe Ratio)?
How did performance vary quarter by quarter?
Which months were strongest and weakest for each stock?
How correlated are these stocks with each other?
Is there a clear trade-off between risk and return?


Key Findings

MSFT had the highest Sharpe Ratio — strongest risk-adjusted return in the group
TSLA was the most volatile stock with the widest monthly swings
Q4 was the strongest quarter across most stocks
Higher volatility did not consistently translate into higher returns
All five stocks show moderate-to-high correlation, limiting diversification within this group

Python Analysis — How to Run

Clone the repo and navigate to the folder

bash   git clone https://github.com/liasaiyan/stock-market-analysis.git
   cd stock-market-analysis

Install dependencies

bash   pip install pandas numpy matplotlib jupyter

Launch the notebook

bash   jupyter notebook stock_analysis.ipynb

Run all cells — data is generated automatically, no external files needed
